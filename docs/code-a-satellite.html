What is a satellite, and why do we care?

A satellite is an object in space, that orbits (travels around) another object, usually a planet such as Earth.

There are many types of satellites, such as:

  • Navigation satellites, e.g. GPS
  • Communication satellites, e.g. Sky TV
  • Natural satellites, e.g. the Moon
  • Space stations, e.g. the ISS
  • Remote sensing satellites, e.g. the one used to take the image above

Remote sensing satellites are used to gather information about an object while orbiting around it. Around the Earth, they are used for a variety of purposes, including weather forecasts, land surveying and humanitarian applications.

In this workshop, you will use a BBC micro:bit to model a satellite, thinking about its power and its orbit, and how it communicates with people back on Earth.

Getting Started

Image from micro:bit

In a new tab, go to the Micro:bit Python Editor.

Before changing any code, get your micro:bit connected:

  1. Plug the micro:bit into your computer
  2. Click the purple Send to micro:bit button on the bottom left
  3. Follow the onscreen instructions to pair your micro:bit

Does your physical micro:bit scroll 'Hello' and show you a heart?

If yes, well done! You can start changing the code - see below for some ideas.

If no, try again, then see the micro:bit user guide for some troubleshooting ideas.


from microbit import *

while True:
    display.show(Image.HEART)
    sleep(1000)
    display.scroll()

Try out the ideas below. It's a good idea to change one thing at a time - test your changes by clicking Send to micro:bit again.

  1. Change the HEART image to something else - watch out for capital letters!
  2. Change 'Hello' to some other text such as your name - be careful of the apostrophes!
  3. What happens if you change the number inside sleep(1000) to something else?
  4. while True means forever - can you change the code so that the scrolling text only happens once at the start instead of repeating?
  5. Try adding sound effects! Can you find the correct syntax in the Reference menu?

Radio Ground Control

It is important that your satellite has a way of communicating with the scientists back on Earth.

The micro:bit at the front is Ground Control - it is set up to receive the messages that you send and display them.

To set this up for yourself at home, click here for the Ground Control code.


To send your messages, you first need to set up the radio, then send your message.

Add the new lines of code starting on line 3

  • After : from microbit import *
  • Before : while True
import radio
radio.on()

Now, pick a name for your satellite - use something short to leave space for error messages. There is a limit of 29 characters, a longer string will be cut short and not displayed by Ground Control.

Then on the next line, use this syntax to send Ground Control a message:

radio.send('PyConUK connected')

Now that you have a way to communicate with Ground Control, you can start sending useful messages.

In the next two sections, you will get data from the micro:bits sensors and check that they are as you expect. If they are not, you can let Ground Control know!

Check Solar Power

Image ©ESA ESA/Mlabspace

The display of the micro:bit is also its light sensor. You will use this to check how much light the micro:bit can sense.

Before you can send useful information to Ground Control, you need to work out what is a 'normal' light level for your satellite.


Inside your while True loop, add this code:

    light = display.read_light_level()
    print(light)

Click the Show serial button on the bottom right - this will open the terminal like this:

What are the light levels like in your area?

  • On the table?
  • When you are holding it?
  • Under the table?
  • When you shine a torch on it?

If your program is running too slowly, delete some of the other code inside your while loop!


If your micro:bit's normal light levels, when sat on the table, are about 100, you can send a warning to Ground Control when it gets too low, using code like this:

    if light < 80:
        radio.send('PyConUK low power')

Challenge: At the moment, you don't care if there is too much light, you only care if the satellite is running out of power. Can you add further code to send a warning when the light levels are much higher than normal?

Check Orbit

Image ©ESA

Another important thing to warn Ground Control about, is if your satellite's orbit is not stable. We can check this by sensing the acceleration.

When a satellite is orbiting the Earth, it is travelling at a constant velocity, meaning that the only acceleration it feels is that of gravity.

  • If it starts to speed up, it will exit the atmosphere and fly off into space - useful for rockets.
  • If it starts to slow down, it will get pulled back down to earth by gravity - useful when the satellite has reached the end of its life.

Using similar code to the light sensing, you can see what acceleration the micro:bit senses when it is not moving:

Inside your while True loop, add this code:

    acc = accelerometer.get_strength()
    print(acc)

Click the Show serial button on the bottom right to see the data.

What are the acceleration levels like?

  • On the table?
  • When you pick it up?
  • When you knock the table?

Can you work out how to send warning messages for an unstable orbit? Look carefully at your light sensing code for a hint.
