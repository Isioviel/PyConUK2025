Getting Started

In a new tab, go to the Micro:bit Python Editor.

Before changing any code, get your micro:bit connected:

  1. Plug the micro:bit into your computer
  2. Click the purple 'Send to micro:bit' button on the bottom left
  3. Follow the onscreen instructions to pair your micro:bit

Does your physical micro:bit scroll 'Hello' and show you a heart?

If yes, well done! You can start changing the code - see below for some ideas.

If no, try again, then see the micro:bit user guide for some troubleshooting ideas.

Try out the ideas below. It's a good idea to change one thing at a time - test your changes by clicking 'Send to micro:bit' again.

  1. Change the HEART image to something else - watch out for capital letters!
  2. Change 'Hello' to some other text such as your name - be careful of the apostrophes!
  3. What happens if you change the number inside sleep(1000) to something else?
  4. while True means forever - can you change the code so that the scrolling text only happens once at the start instead of repeating?
  5. Try adding sound effects! Can you find the correct syntax in the Reference menu?
